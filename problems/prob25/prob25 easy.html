• Boolean Tree

A word m is a finite sequence m0 m1 . . . m l−1 of letters,of the alphabet or {0,1}. Its length l is denoted l(m) and its letter of index i is denoted mi for 0 ≤ i ≤ l(m)−1. The empty word, of length 0, is denoted ε.
A binary tree is defined recursively as follows:
• either like the empty tree, noted V , which does not contain any node.
• either as a node, denoted N(x, g, d) and called root, where x is the information stored in the node, g is a binary tree called left subtree and d is a binary tree called right subtree.
A word tree is a binary tree in which each node contains a boolean. A tree of words a represents a finite set of words, denoted M(a), defined as follows. If the tree a is empty, then M(a) is the empty set. If the tree a is of the form N(b, a0, a1), then M(a) is the set of the following words:
• the empty word ε if the boolean b is true;
• words of the form 0m for m ∈ M(a0);
• words of the form 1m for m ∈ M(a1).
In other words, M(a) is the set of words corresponding to the paths leading from the root to a node containing true, using the character 0 when going left and the character 1 when going right.

Input:
**[root[left child,right child],...]
Output:
**Set of words corresponding to the paths leading from the root to a node.

Input Output
a=[T[,F[T,T]] M(a)={ε, 10, 11}
