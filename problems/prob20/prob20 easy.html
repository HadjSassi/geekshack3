• 4-game

Connect 4 is a strategic two-player board game played on a n*m grid (n>6;m>6). The grid consists of vertical columns and horizontal rows. Each player is assigned a unique color, typically red or yellow. The objective is to connect four of one's own discs of the same color consecutively in any direction: horizontally, vertically, or diagonally.
First, the game starts with a pre-filled grid as input. The objective is to check if there is a winner or not. If there is no winner, the current player makes their last move, then a final check is made to determine the final result.

Input:
**Giving two space seperated integers, first is the number of the rows and the second is the number of the columns of the grid.
In the input we find the grid filled with 0,1 and 2. Zero for the empty places, 1 for the red color and 2 for the yellow color.

Output:
1 : if the first player is the winner.
2 : if the 2nd player is the winner.
0 : otherwise.


Constraints:
** Time Limit : 5s

  • Notes
      The player using the red color always initiates the game.
      You can only substitute a 0 for 1 or 2 if there is no 0 below it.
