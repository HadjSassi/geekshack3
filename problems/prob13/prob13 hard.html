• Printer Machine Purchase

A printing company has a budget of B to buy production machines.
There are N types of machines. Each type i of machine has a price Pi and an efficiency Ri.
The company wants to maximize the total output (number of runs/min) obtained without exceeding its budget B.
Note that it is possible to buy 0, 1 or 2 machines of the same type.
The inputs of the problem will be as follows:
B
N
Pi Ri ; 1 <= i <= N
____
B : Budget
N : Number of machine types
Pi : Price of a machine
Ri : efficiency of a machine
It is necessary to make sure that if we are going to choose a certain machine the constraint to be checked is that SUM(xi*Pi) <= B ; xi= {0,1,2}
Here is an example:

200
3
80 70
60 50
40 25
Your code must return 
170
This means that the company buy two machines of type 2 and only one machine of type 1 ( 70*1 + 50*2 ).
It gives the best efficiency where there prices does not exceed the budget.
