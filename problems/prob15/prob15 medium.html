• Special Square

A special square is a matrix of order 2 composed of 4 integers (1, 2, 3 and 4).
The aim of this exercise is to determine the number of special squares in a matrix.
Write a program that solves this problem.

Input:
The first line contains the number of rows N of the given matrix. The next N rows, each of which contains one row of the matrix

Output:
You should have a line containing the number of special squares in the given matrix.

NOTES

  • EXAMPLE :
Input Output
3
1 2 1 3
4 3 4 4
2 1 1 1
 2
