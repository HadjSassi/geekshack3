In this statement we will simulate the process of merging of git branches info main branch.
Here an illustration to explain the process:

Here we have two branches, A and B , each one of them has its own data
The main Branch had already, stored data
Merge process will push inexsited data or modify incompleted data
For exemple, tha branch A contains "A B C D" as part of data where "C" lettre does not exists in Main data branch
So we have to modify the "A B D" data and push it to Main to get "A B C D : M". and for inexisted data like "H I K" in branch B will be pushed to Main and we will have new Uploded data like this "H I K : U"
* M: means Modified
* U: means Uploaded
We will have as input for this exemple: 

    5
    BR Main A B D
    BR Main 5 3 2
    BR A A B C D
    BR A 5 3 2 
    BR B H I K
First input line is the nubmer of input data
* BR: means Branch
* A or B: the name of branch
OutPut must be like :
    A B C D : M
    H I K : U
    5 3 2
