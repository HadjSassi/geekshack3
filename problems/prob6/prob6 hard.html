• Escape Room

The wizard locked you in a magic room. No much time left and you want to open the door by solving its pattern. Luckily, you have X gems that will help you escape. The door is illustrated by a matrix, each column contains a number in [0,8] or the symbol '+'. + : The right place of a gem. 0..8 : Sum of gems in the neighbour columns. To escape you should put the gems in the right places and complete the matrix by the missing numbers.

Input:
**N : Size of the matrix in [4..100] (it's a square matrix).
**X : Number of the gems in [3..70].
**M : The matrix
Output:
**The matrix M completed ( if it's not possible display : Not Valid ).

  • Example :
Input Output
4
4
[1,-,2,-]
[1,-,3,-]
[-,-,-,1]
[-,2,1,1]
[1,1,2,+]
[1,+,3,2]
[2,3,+,1]
[+,2,1,1]
  • Notes
      All the gems are identical.
      An empty column is marked by "-".
      If empty columuns > filled columns, display Not Valid.
      In corners, there are only 3 neighbours.
      On edges, there are only 5 neighbours.
