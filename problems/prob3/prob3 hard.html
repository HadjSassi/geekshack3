• Solve Me

You have a string with missing characters and a list of characters, replace each space (" "). with a character from that list.
The final completed string has a score which is calculated as following :
The value of each character is the sum of its index in the alphabet and its index the string.
The score the sum of character values.
A certain number will be shown, you should tell if it is possible to get exactly that score by making the appropriate combination for its string.

Input:
** 1st input is the string
** 2nd input is a list of characters
** 3rd input is the score
Output:
** Message: Valid/Not Valid

  • Example :
Input Output
   e
[a,b,c,d]
13 		Valid
  • Notes
      The index starts with 0
      All characters are lowercase
      You can use the same character more than once
      if it's not possible to reach the score display Not Valid
      You can't change the position of the characters that are already in the given string
