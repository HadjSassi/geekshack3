• Water Economics - II

As a part of finding solutions around saving water, "Sonede" is calculating the time of use of push button faucets for a house.
Each such faucet delivers water for N seconds. If this button is pushed while water is flowing, the faucet will flow for another N seconds.
*But in this statement we have multiple user.
The inputs to this problem are multiple lines:
The 1st line contain the number of users
The N*2 other lines contains a couple of data with describe pushing number and faucet water deliver time(N) in the first data.
The second data contains the time at which the user of the faucet pushes the button again.
Example:

    2
    2 4
    0 3
    2 4
    0 5
Your solution must return the sum of tims of each use
For example, your code must return 15 ( 7(for the first user) + 8 (the second user)) the Output must be :
    15

NOTES

You should assume the input exeption ! If input contain caracters, your code must ruturn 'NO' as output.

Output must be an integer.

No spaces in the beginning and the end output.
